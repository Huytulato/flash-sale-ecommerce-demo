# Flash Sale Ecommerce Demo

This project demonstrates a simple Flash Sale system for an e-commerce platform

## Tech Stack

Backend
- Java
- Spring Boot

Frontend
- React

## Project Structure

flash-sale-ecommerce-demo
 ├── backend
 ├── frontend
 └── docs

## Run Backend

cd backend
./mvnw spring-boot:run

## Run Frontend

cd frontend
npm install
npm start
