# Flash Sale Ecommerce Demo

This project demonstrates a simple Flash Sale system for an e-commerce platform

## Tech Stack

Backend
- Java
- Spring Boot

Frontend
- React

Database & Cache: 
- PostgreSQL
- Redis

## Project Structure

flash-sale-ecommerce-demo
- backend
- frontend
- docs

## Start Database & Redis
docker-compose up -d

## Run Backend

- cd backend
- mvn spring-boot:run

## Run Frontend

- cd frontend
- npm install
- npm run dev
